# Balloon Popper

Tracks balloons popped in the Party Room and displays them as custom Party Skill XP (replacing the Sailing skill label).

## Features
- **Dynamic XP Tracking**: Earn "Party XP" by popping balloons.
- **In-Game Sidebar**: Access a custom sidebar with real-time stats and a toggleable in-game guide.
- **Collection Log**: Track every item you've ever received from a balloon, with automatic normalization for noted items and quantity tracking.
- **Level Up Effects**: Standard OSRS level-up fireworks, sound effects, and congratulatory messages/dialogs.
- **Modern UI**: Polished, responsive design that fits the RuneLite aesthetic.
